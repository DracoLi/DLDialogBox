Overview

DLChoiceDialog is a dialog that asks for user input from a list of choices.

DLChoiceDialog provides a simple way for you to gain user input without writing up much code.

A DLChoiceDialogCustomizer is used to customize the choice dialog.

Properties

choices

An array of strings that contains the choices for this choice dialog.

@property (nonatomic, copy) NSArray *choices

Discussion

Setting this array will actually redraw/reposition the choice dialog’s content according to the current customizer.

However it is still recommended to not change this array once your choice dialog is created.

Declared In

DLChoiceDialog.h

customizer

This customizer is used to customize the UI and functionalities of the choice dialog.

@property (nonatomic, strong) DLChoiceDialogCustomizer *customizer

Discussion

Attempting to update any UI related properties in the customizer will not do anything and may break some functionalities.

You can however update functionality related properties on the customizer as those are processed whenever they are required.

selectChoiceAtIndex:skipPreselect:

Programatically selects a choice by passing the index of the choice in the choices array.

- (void)selectChoiceAtIndex:(NSUInteger)index skipPreselect:(BOOL)skipPreselect

Parameters

index

Index of the choice. Starts at 0.

skipPreselect

If set to YES, this method will bypass any preselects.

Discussion

If skipPreselect is set to NO, this method will preselect the label first if is enabled and the targeted choice has not already been preselected.

Note: The delegate will be notified of this selection (maybe be a preselect or select).

Note: index starts at 0.
