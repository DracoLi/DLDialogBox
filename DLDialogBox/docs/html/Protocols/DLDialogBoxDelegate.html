dialogBoxAllTextFinished:

Called after all the text has been displayed by the dialog box.

- (void)dialogBoxAllTextFinished:(DLDialogBox *)sender

Discussion

If the dialog box needs to open a choice dialog after the last text then this is called after the choice dialog is displayed, not after a choice has been selected.
