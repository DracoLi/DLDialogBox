Properties

backgroundColor

If a sprite is not provided as the dialog’s background, this property will be used as the background color of the dialog box.

@property (nonatomic) ccColor4B backgroundColor

Discussion

Defaults to a transparent color (ccc4(0,0,0,0)) if not specified.

You can create a ccColor4B via ccc4(red, blue, green, alpha). Note that all color values are from 0-255.

backgroundSpriteFile

The file name of a stretchable sprite image that will be used as the background image for the dialog box.

@property (nonatomic, copy) NSString *backgroundSpriteFile

Discussion

If the backgroundSpriteFrameName is also provided, then this value will be ignored.

Please refer to the usage documentation on how the sprite image should be made.

backgroundSpriteFrameName

The sprite frame name of a stretchable sprite image that will be used as the background image for the dialog box.

@property (nonatomic, copy) NSString *backgroundSpriteFrameName

Discussion

If a backgroundSpriteFile is also provided, then only this value will be used.
